# Brooke's Wisconsin Garden

A mobile-first Wisconsin gardening companion built for Green Bay. The first version includes:

- A polished **Today** dashboard with local weather and garden tasks
- A **What Can I Plant Now?** guide tailored to Green Bay / Zone 5b
- A **My Garden** overview for raised beds, greenhouse, and hydroponics
- Interactive task completion, filters, detail drawers, and saved local state
- Installable PWA metadata and an offline-ready service worker
- Responsive layouts tested at 320px, 390px, and 430px widths

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

Render settings:

- Build command: `npm install && npm run build`
- Publish directory: `dist`
- Branch: `main`
