# Brooke's Wisconsin Garden

A mobile-first, year-round Wisconsin gardening companion built for Green Bay.

## Working systems

- Live Green Bay weather and automatic year-round seasonal priorities
- Actionable Garden Briefing and task snoozing
- Reorderable, hideable, removable garden spaces
- Detailed plant control center: edit, move, choose stage, notes, archive, or delete
- Measured harvest logging with amount, unit, quality, notes, and grow-again preference
- Problem records for pests, disease concerns, weather damage, animal damage, and plant loss
- Greenhouse temperature and humidity reading history
- Seven-day and thirty-day planning, succession planting, and seed inventory
- Persistent local garden history across outdoor, greenhouse, hydroponic, indoor, and basement spaces
- Round cheese sun with a tiny WI Grown farm badge
- Brighter cream, cheese-yellow, green, brown, and red cheese-sprout app icon
- Installable PWA behavior with refreshed offline cache v6

The collapsible garden-card concept remains reserved for the later visual-overhaul phase.

See `ROADMAP.md` for the remaining camera/QR, deeper seed controls, environment control centers, shared sync, and garden-planning phases.

## Render deployment

- Build command: `npm install && npm run build`
- Publish directory: `dist`
- Branch: `main`

Every new commit to `main` should trigger an automatic deployment.
